## â­ What is a List?

A list is a Python data structure used to store multiple values in a single variable.
Lists are created using **square brackets `[ ]`**.

Example:

```python
fruits = ["apple", "banana", "cherry"]
numbers = [10, 20, 30, 40]
```

---

## â­ Properties of Lists (Very Important for Interviews)

### âœ” 1. Mutable

* List elements **can be changed** after creation.

```python
nums = [10, 20, 30]
nums[1] = 99   # â†’ [10, 99, 30]
```

### âœ” 2. Allow Duplicates

* Same value can appear multiple times.

### âœ” 3. Ordered

* Elements maintain the order in which they were inserted.
* You can access values using **indexing**.

### âœ” 4. Heterogeneous

* A list can store **multiple data types**:

```python

```

## â­ Indexing & Slicing (Same as Strings)

### â¤ Indexing

```python

nums[0]    # 10
nums[-1]   # 30
```

### â¤ Slicing

```python

nums[1:]    # [20, 30]
```

# â­ List Traversing

Ways to traverse a list:

### âœ” Using index

```python
for i in range(len(nums)):
    print(nums[i])
```

### âœ” Directly

```python
for x in nums:
    print(x)
```

# â­ Important List Methods (Must Know for Placements)

Given:

```python
numbers = [5, 2, 9, 1, 5, 6]
```

| Method         | Description             | Example                    |
| -------------- | ----------------------- | -------------------------- |
| `append(x)`    | Add at end              | `numbers.append(10)`       |
| `insert(i, x)` | Insert at index         | `numbers.insert(2, 15)`    |
| `extend(list)` | Add multiple items      | `numbers.extend([20, 25])` |
| `remove(x)`    | Remove first occurrence | `numbers.remove(5)`        |
| `pop(i)`       | Remove & return element | `numbers.pop(3)`           |
| `index(x)`     | Return index of x       | `numbers.index(6)`         |
| `count(x)`     | Count occurrences       | `numbers.count(5)`         |
| `sort()`       | Sort ascending          | `numbers.sort()`           |
| `reverse()`    | Reverse the list        | `numbers.reverse()`        |
| `copy()`       | Creates a shallow copy  | `new = numbers.copy()`     |
| `clear()`      | Remove all elements     | `numbers.clear()`          |
